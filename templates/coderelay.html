Code Relay Challenge

Team Composition

4 players per team

Game Rules

  • Each team will be given a coding problem
  • Only one member can code at a time (5-minute limit per coder)
  • Next member continues from where previous coder left off
  • No verbal communication during handovers
  • No internet searches or AI assistance
  • Skipping turns or breaking format results in penalties

Judging Criteria

  • Correctness of solution
  • Code efficiency
  • Readability
  • Time taken

Event Structure

  • Number of Rounds: 3
  • Time per Round: 30 minutes
